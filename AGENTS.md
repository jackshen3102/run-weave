# AGENTS

面向编码智能体的高层路由与最小默认行为。

## 项目概览

- 前端：React + Vite
- 后端：Express + WebSocket + Playwright 控制
- 共享协议：packages/shared

## 最小命令

- 开发：`pnpm dev`
- 构建：`pnpm build`
- 类型检查：`pnpm typecheck`
- Lint：`pnpm lint`
- 测试：`pnpm test`
