# 原型标题

面向 `<feature>` 的可运行 HTML 交互原型。

## 启动

```bash
python3 -m http.server 6188 --directory docs/prototypes/<feature-slug>
```

打开：

```text
http://127.0.0.1:6188/
```

## 文件

- `index.html`：静态页面壳层、样式和挂载点。
- `app.js`：原型状态流转和事件处理。
- `mock-state.json`：模拟领域对象、UI 状态和事件流。
- `prototype-preview.png`：可选的验证截图。

## 边界

- 这个原型不连接真实后端 API。
- 这个原型不导入生产源码。
- 这个原型不能证明产品协议、存储或运行时支持已经存在。
- 原型专用控件只是验证脚手架，不是最终产品 UI。
- 原型辅助功能默认不进入实施计划，除非在冻结记录中明确转为产品需求。
