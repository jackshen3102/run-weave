# architecture-flows（可运行架构流程）

基于真实代码、接口和运行证据制作的可运行 HTML 技术说明，用于讲清系统关系、事件流、状态流、运行闭环和故障因果，并从流程中发现值得验证的潜在问题。

- **与产品原型分离**：本目录只放技术架构流程；页面、组件和产品交互原型放在 `docs/prototypes/`。
- **与活架构文档分离**：稳定的当前架构事实沉淀到 `docs/architecture/`；本目录保留特定代码基线下的可运行说明和诊断证据。
- **事实边界**：每个子目录通过 README 记录代码基线、来源、结论和非目标；HTML 中的导航或筛选不代表产品 UI。
- **默认预览**：按子目录 README 提供的命令启动本地 HTTP server，不直接通过 `file://` 打开。
