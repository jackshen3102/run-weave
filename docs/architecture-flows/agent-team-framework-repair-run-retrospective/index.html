目标架构中的 Human Gate

只保留需要人类价值判断的门禁

AUTO RECOVER

机械可恢复

  • thread busy
  • 重复 completion
  • blocker 已解除
  • 合同字段可补交
RETRY / FAIL

系统故障

  • 持久化失败
  • 目标 pane 消失
  • 超时后仍不可投递
  • fixture cleanup 失败
HUMAN DECISION

语义裁决

  • 范围是否扩张
  • P0/P1 finding 是否接受
  • 验收合同是否修改
  • 风险是否延期
